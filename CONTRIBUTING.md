# Contributing to BrainVION

Thank you for your interest in contributing to BrainVION.

BrainVION is a student technology community focused on learning,
collaboration, and building real-world technical skills.

We welcome contributions from students, developers, designers,
and community members.

---

## How to Contribute

1. Fork the repository
2. Create a new branch for your feature or fix
3. Make your changes
4. Submit a Pull Request

Example:

git checkout -b feature/homepage-ui

---

## Contribution Guidelines

Please follow these rules when contributing:

• Write clean and readable code  
• Keep components modular  
• Use descriptive commit messages  
• Test your changes before submitting  

Example commit messages:

feat: add homepage hero section  
fix: mobile navigation issue  
docs: update README  

---

## Code Standards

• Follow consistent naming conventions  
• Keep files organized in the correct folders  
• Avoid unnecessary dependencies  

---

## Community

BrainVION is a student-first community.  
Respectful collaboration and positive communication are expected.
